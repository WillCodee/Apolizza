# Apolizza - Corretora de Seguros

Site profissional e responsivo para Apolizza Corretora de Seguros, desenvolvido com HTML, CSS e JavaScript puro.

## 🎨 Características

- **Hero dinâmico** com imagem de família como fundo
- **Logo estilizada** com tipografia Nunito
- **Navegação responsiva** com menu mobile
- **Seções completas**:
  - Sobre nós com missão e valores
  - Equipe com fotos
  - Seguros por categoria (Patrimonial, Saúde/Dental, Vida)
  - Seguradoras parceiras
  - Formulário de cotação
  - Mapa de localização
  - Footer com contatos

## 🔧 Tecnologias

- HTML5
- CSS3 com design responsivo
- JavaScript vanilla
- Google Fonts (Poppins, Nunito)
- Clearbit API para logos de seguradoras

## 📦 Como usar

1. Clone o repositório
2. Abra `index.html` no navegador
3. Ou suba em um servidor HTTP:

```bash
# Com Node.js
npx http-server

# Com Python
python -m http.server 8000

# Com PHP
php -S localhost:8000
```
