# Apolizza - Corretora de Seguros

## 📋 Sobre o Projeto

Site profissional e moderno da **Apolizza Corretora de Seguros**, uma empresa baiana com mais de 10 anos de experiência no mercado de seguros, oferecendo consultoria humanizada para pessoas físicas e jurídicas.

## 🎯 Funcionalidades

- **Homepage moderna** com hero section impactante
- **Seção Sobre** - Informações sobre a empresa
- **Missão, Visão e Valores** - Cultura organizacional
- **Equipe** - Apresentação das diretoras com fotos
- **Benefícios** - Por que escolher Apolizza
- **Seguros** - Catálogo de produtos com 3 categorias:
  - Patrimonial / Auto
  - Saúde / Dental
  - Vida PF e PJ
- **Parceiros** - Logos de 31 seguradoras
- **Contato** - Formulário de cotação e informações
- **WhatsApp integrado** - Acesso direto via flutuante
